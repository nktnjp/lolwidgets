# LOLWidgets
## 概要
LOLWidgetsはLeague of Legends(LOL)のゲーム情報をウィジェットに表示することにより、
ユーザーが迅速に情報を得ることをサポートするiOSアプリケーションです。<br/>
初回リリース時の機能は「無料チャンピオン一覧」「サモナー一覧」の2つです。<br/>
対応している地域はJPとNAで、設定画面から変更することが可能です。<br/>
ただし、言語は日本語のみです。<br/>
